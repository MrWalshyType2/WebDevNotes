Symbols in HTML

Often, there are symbols in every day use that we do not find on the standard QWERTY, AZERTY, etc... keyboards. To compensate for this, HTML provides a variation of codes that can be used to represent them.

Find a full table of symbols in HTML here.

HTML Character Entities

It is commonplace when writing content in a HTML document that certain reserved symbols are required, how do we get the < symbol to display for example. Character entities are short codes that can be written to display such characters. Some examples include:

  • &lt; to display the less than symbol.
  • &gt; to display the greater than symbol.
  • &amp; to display an ampersand.
  • &pound; to display the British pound symbol.
  • &copy; to display the copyright symbol.
  • &reg; to display the registered trademark symbol.

Other symbols and forms of representation

Not every symbol available in HTML5 has a named character entity code, but instead one or more of the following:

  • HTML code, like &#9824; for the spade symbol: ♠
  • Hex code, like &#x2660; for the spade symbol: ♠
When researching Unicode symbols, they will generally be of the form U+02660. Simply replace the U+ with &#x and put a semicolon on the end to make it a usable hex code character entity in HTML. So U+02660 would be &#x02660; or &#x2660; as we can also drop any leading zero's from hexadecimal values.
