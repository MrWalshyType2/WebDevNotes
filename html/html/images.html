Images

General

In HTML documents, images can be embedded using the self-closing img tag to reference the image - whether locally stored or on the internet. The img element has only one mandatory attribute, the src attribute which holds a path or URL to the desired image to display. For example:

<img src="./images/image.png" />

This example uses a relative path from the current project root, you may also change this for a URL to an image:

<img src="https://www.qa.com/media/14650/qa-logo-white-on-blue.png?anchor=center&mode=crop&width=640&height=500&rnd=132307537440000000" />

Although the src attribute is the only mandatory attribute for image elements, we should also almost always treat the alt attribute - an attribute for specifying alternate text describing an image for assistive-technologies - as mandatory. This ties into legal requirements for making websites accessible under the Equality Act 2010 and the Public Sector Bodies (Websites and Mobile Applications)(No.2) Accessibility Regulations 2018. Just know that specifying an alt attribute with appropriate text describing the image helps to meet these legal obligations. The following example demonstrates using the alt attribute to describe the QA logo:

<img alt="The letters QA in white on a blue background" src="https://www.qa.com/media/14650/qa-logo-white-on-blue.png?anchor=center&mode=crop&width=640&height=500&rnd=132307537440000000" />
We should almost always specify the alt attribute, but cases where you might not use this attribute include images non-essential to the content of the page - in which case, set the attribute to an empty string: alt=""

Image attributes

Images have multiple attributes which can be used to control how they are presented, some of these attributes are as follows:

Attribute Description
width Sets the width of the image in pixels.
height Sets the height of the image in pixels.
alt Sets alternative text for use by screen-readers, and other accessibility technologies, alongside being displayed when an image cannot be loaded.
src Sets the source of an image to load from, whether local or remote.

The width and height of an image can be controlled using the attributes described above, for example:

<img width="320" height="250" alt="" src="" />

It is preferable to control the width and height of an image using CSS properties instead, the following example makes use of the style attribute to allow CSS to be supplied as the value:

<img style="width: 320px; height: 250px" alt="" src="" />

Images as links

Images can be turned into links by nesting them inside of an anchor element, for example:

<a href="#">
    <img style="width: 320px; height: 250px" alt="" src="" />
</a>

Whatever image is loaded and placed where the image element is defined will be turned into a clickable link as it is wrapped in an anchor tag.

When href="#" is supplied to an anchor tag, the element will still appear as a link but it will not navigate anywhere.
