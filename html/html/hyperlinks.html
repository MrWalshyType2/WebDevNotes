Hyperlinks

In HTML documents, it is often desirable to link to other documents from the current one - whether stored and accessed on the local machine or via the web.

Anchors

To create links to other documents in the current document, an a tag (short for anchor) must be supplied. The a tag has two parts:

  • Hyperlink: A link specified in the href attribute to another web document, this is not displayed to the user.
  • Hypertext: The text within the anchor elements tag that is displayed to the user.

To create a link to another document, specify the anchor tag like the following - replacing the hyperlink and hypertext as needed:

<a href="https://www.qa.com">QA</a>

To reference a locally stored document instead, use a relative path in the href attribute rather than a URL to a document on the web:

<a href="/about.html">About us</a>

Targets

By default, anchor elements will open the specified document in the current tab when pressed. This may not always be desirable, and thus can be changed using the target attribute on an anchor element. The possible target values are as follows:

Target Description Example
_blank Opens the specified hyperlink in a new tab. <a href="https://www.google.com" target="_blank">Google _blank</a>
_self Opens the specified hyperlink in the current tab (default). <a href="https://www.google.com" target="_self">Google _self</a>
_top Opens the specified hyperlink the top most frame, or the current tab if already in the top most frame (see iframes). <a href="https://www.google.com" target="_top">Google _top</a>
_parent Opens the specified hyperlink in the parent frame/document of the current frame (see iframes). <a href="https://www.google.com" target="_parent">Google _parent</a>
_framename Opens the specified hyperlink in the specified frame. <a href="https://www.google.com" target="some_frame">Google _parent</a>

Mailto attribute

The href attribute is capable of taking a mailto value which opens the default mail client with the supplied values, for example:

<a href="mailto:somemail@emailcom?subject=Hello World&body=Hello there sir/madam">Send email</a>

This will open the default mail client on the users computers and prefill it with the following information:

  • A recipients email specified after mailto:
  • A subject and a body specified as key=value pairs after the ? symbol with each query parameter separated by an & symbol
