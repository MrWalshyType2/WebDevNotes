The document head

An area oft-used for supplying page metadata and linking in stylesheets

Metadata

Metadata is data used to describe other data, in the case of HTML we use metadata to describe additional information concerning our HTML document.

Metadata is often used to specify things like:

  • The character set of the document
  • A description
  • Keywords
  • Document authors
  • Viewport settings

The meta element

The meta element is used to describe the metadata of a document, this element is special in that it must be placed within the head element of the document, meaning it will not be rendered to the user.

The following example demonstrates using some of the core meta attributes:

<head>
    <!-- Set the character set -->
    <meta charset="UTF-8" />
    
    <!-- Set keywords for search engines -->
    <meta name="keywords" content="HTML, CSS, JS" />
    
    <!-- Set the documents description -->
    <meta name="description" content="tutorial of HTML" />
    
    <!-- Set the documents author -->
    <meta name="author" content="John Smith" />
    
    <!-- Set the viewport width -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
</head>

The last meta element specified is an important one, this ensures that the HTML elements in our document scale appropriately according to the devices size that is viewing the document:

  • width=device-width sets the documents width to match that of the device viewing the document
  • initial-scale=1.0 sets the zoom level of the document
